## ESP32 Custom Wi-Fi Library

### *Feature:*
        1. Change Wi-Fi credentials dynamically with web interface and serial command
        2. Auto redirect to Wi-Fi setup page(no need to know/type ip on web browser)
        3. Easy to use 
        4. No extra dependency (use default esp32 library(s)

### *Usages:*
        1. Include this library into your project source code 
        2. Setting up server with port
